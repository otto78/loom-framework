Quick Start

Get started in 5 minutes — 3 ways to set up Loom:

① Natural Language (easiest)

Just tell your AI agent:

"setup loom framework"

② Interactive Wizard

git clone https://github.com/otto78/loom-framework.git cd /path/to/your-project python /path/to/loom-framework/loom/scripts/setup.py

③ Automated (no prompts)

python loom/scripts/setup.py --auto --project-name "MyProject" --ide windsurf,cursor

Then use natural language with your agent:

"start task TASK-001 'implement user login'" "list tasks" "run tests" "complete task TASK-001"
